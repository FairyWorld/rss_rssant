时光如梭，打磨了整整一年，蚁阅终于迎来正式版！

主要特点：

- 非社交，无广告，无推荐，专注阅读
- 为移动端优化，适合随时随地阅读
- 按订阅更新频率区分消息，好文章慢慢看，读资讯一目十行
- 输入博客地址，智能查找订阅，支持批量导入导出
- 智能图片代理，解决图片无法加载问题
- 开源，可以自己部署，也可以直接用在线版

开箱即用地址: [https://rss.anyant.com](https://rss.anyant.com) (建议用手机浏览器访问)
